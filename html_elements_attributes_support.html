HTML DOM elements and attributes support

The list below is not one of current trends or statistics, but support for this particular browser version.

Elements include those from the Jurassic Era to HTML5, and the attributes include the latest mobile device innovations if a mobile device is detected.

Information listed below is obtained by querying the HTML API to discover the status of currently supported and daprecated attributes.

Columns for each table are: element, if supported (graphic), the actual API reference returned, the element-specific properties, and lastly other attributes found i.e. non-standard, vendor-specific, microdata and global

Element legend (row colours):

  • <a> - element is supported
  • <listing> - element is not supported
  • <basefont> - unusual or generic reference (may have different property support), normally the parent (HTMLElement). Hover over the name if followed by an asterisk to see which value was expected.

Attributes legend:

  • rel - attribute is supported ping 5 - attribute is new in HTML5
  • href - attribute is required
  • datasrc - attribute is not supported by this browser
  • coords - attribute is deprecated, but still supported by this browser
  • charoff - attribute is obsolete but still supported
  • title - global attribute
  • volume - attribute is non-standard or browser-specific
